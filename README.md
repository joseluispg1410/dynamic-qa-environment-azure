# Dynamic QA Environment on Azure

This project demonstrates how to design and implement a dynamic QA environment on Azure using Infrastructure as Code, CI/CD pipelines, and automated testing.

## Goals
- Provision QA environments on demand
- Deploy an application automatically
- Execute automated UI and API tests
- Destroy all resources after execution
- Ensure cross-platform compatibility

## Tech Stack
- Azure
- Terraform
- GitHub Actions
- Playwright
- Docker
